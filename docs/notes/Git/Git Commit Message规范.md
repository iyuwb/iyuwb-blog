# Git Commit Message规范

# 代码初始化 `init`

```shell
git commit -m 'init:项目初始化'
```

## 新功能 `feat`

```shell
git commit -m 'feat:添加新功能'
```

## 修复BUG `fix`

```shell
git commit -m 'fix:修改BUG'
```

## 重构 `refactor`

```shell
git commit -m 'refactor:下载方法重构'
```

## 样式改动 `style`

```shell
git commit -m 'style:首页样式'
```

## 文档改动 `docs`

```shell
git commit -m 'docs:使用手册'
```
