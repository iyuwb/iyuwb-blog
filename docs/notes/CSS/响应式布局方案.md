# 响应式布局方案
> 根据不同设备或者不同尺寸的屏幕，页面有不同的布局

### 媒体查询 Media Queries
根据不同的屏幕尺寸，设置不同的css样式，通过覆盖样式，实现响应式

```css
@media screen and (max-width: 980px) {
  #head { … }
  #content { … }
  #footer { … }
}
```
