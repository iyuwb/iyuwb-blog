# 移动端H5-点击异常背景色


## 问题复现

如下图，当点击左侧侧边栏时，会出现一个淡蓝色背景
![alt text](image-1.png)

## 问题解决
再出现背景的元素上添加：`-webkit-tap-highlight-color`样式

```css
-webkit-tap-highlight-color:transparent
```
